Sobre mí

Hola, somos el grupo los jausi, estudiantes de ing de sistemas. Este sitio web es parte de un proyecto universitario donde combinamos dos cosas que mas nos gustan: la gastronomía y el diseño/desarrollo web.

El objetivo del blog es practicar:

  • Diseño de interfaces en modo oscuro y organización de contenido.
  • Estructura de páginas HTML, estilos en CSS y algo de JavaScript.
  • Redacción de reseñas claras, honestas y fáciles de leer.

Cada reseña representa una simulación de visita a un restaurante o cafetería, escrita desde el punto de vista de un cliente normal, sin lenguaje técnico de gastronomía, para que cualquiera pueda entenderla.

Enfoque del proyecto

Más que hacer publicidad, el enfoque es académico:

  • Practicar maquetación web con HTML y CSS en modo oscuro.
  • Aplicar JavaScript básico para filtros, botones y navegación.
  • Documentar el proceso como parte de un trabajo universitario.

Cómo leer el blog

Te recomiendo empezar por la reseña principal en la página de Inicio y luego revisar la lista completa en la sección Reseñas. El formulario de Contacto es solo de ejemplo y no envía datos reales.
